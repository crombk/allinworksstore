## 🎯 SEO 기본 전략

### 네이버 쇼핑 검색 알고리즘 이해

네이버 쇼핑 검색 순위는 다음 3가지 요소로 결정됩니다:

1. **적합도 (Relevance)**: 검색어와 상품 정보의 일치도
   - 상품명, 카테고리, 속성값이 검색어와 얼마나 일치하는가
   - 표준상품명 가이드 준수 여부

2. **인기도 (Popularity)**: 상품의 판매 실적 및 고객 반응
   - 판매량, 매출액, 리뷰 수, 리뷰 평점
   - 찜 수, 클릭률 (CTR)

3. **신뢰도 (Trust)**: 판매자 및 상품의 신뢰성
   - 스토어 등급, A/S 이력
   - 반품률, 교환율, 상품 문의 응답률

### B2B 산업용 공구의 SEO 특수성

- **긴 검색어 (Long-tail Keywords)**: "산업용 드릴" 보다 "항공우주용 고정밀 드릴"
- **전문 용어 혼용**: 한글/영문 혼합 검색 빈도 높음 (예: "임펄스 렌치", "impulse wrench")
- **토크 범위/사양 중요**: 숫자 정보 포함 시 검색 정확도 향상

## 📝 상품명 작성 규칙 (네이버 표준 가이드 준수)

### ✅ 반드시 지켜야 할 9가지 규칙

1. **정보 기입 순서** (있는 경우 필수 기입)
   ```
   브랜드/제조사 → 시리즈 → 모델명 → 상품유형 → 색상 → 소재 → 
   패키지수량 → 사이즈 → 성별/나이 → 속성(용량/무게/토크범위)
   ```

2. **50자 이내 권장** (최대 100자 허용)
   - 50자 이상은 어뷰즈로 판단될 확률 높음

3. **브랜드/제조사/상품명 반복 금지**
   - ❌ 나쁜 예: "아트라스콥코 Atlas Copco 아트라스콥코 드릴"
   - ✅ 좋은 예: "아트라스콥코 LBB16 배터리 피스톨 드릴"

4. **셀러명/쇼핑몰명 포함 금지**
   - ❌ 나쁜 예: "올인웍스 아트라스콥코 드릴"
   - ✅ 좋은 예: "아트라스콥코 LBB16 드릴"

5. **한글 우선, 필요시 영문 사용**
   - 숫자는 아라비아 숫자로 표현
   - 한글/영문 외 다른 언어 사용 금지

6. **허용 특수문자만 사용**
   - 허용: ( ) - · [ ] / & + , ~ .
   - 특수문자도 과다 사용 금지

7. **패키지 상품은 내용물 숫자 정확히 표현**
   - ✅ 좋은 예: "아트라스콥코 소켓 세트 24pcs"

8. **이벤트/할인/가격 정보 금지**
   - ❌ 금지: "50% 할인", "무료배송", "1+1", "특가"
   - 별도 이벤트 필드 활용

9. **단일 상품만 판매**
   - 카테고리/유형이 다른 상품을 하나로 묶지 않음
   - 서로 다른 브랜드 상품을 모아 판매 금지

## 🔗 내부 링크 구조 (SEO 최적화)

### 계층 구조

```
홈페이지 (index.html)
├── 제품 카테고리 (category-*.html)
│   ├── 에어 조립 도구 (category-air-tools.html)
│   ├── 배터리 조립 도구 (category-battery-tools.html)
│   ├── 유선 핸드헬드 도구 (category-cable-tools.html)
│   ├── 재료 제거 도구 (category-material-removal.html)
│   ├── 정밀 저토크 도구 (category-precision-tools.html)
│   └── 컨트롤러 & 소프트웨어 (category-controllers.html)
│
├── 상품 상세 (product-detail.html?series=XXX)
│
├── 고객지원 (faq.html)
│
└── 회사소개 (#about)
```

### 내부 링크 전략

1. **Breadcrumb 필수**
   ```html
   홈 > 제품 카테고리 > 에어 조립 도구 > ErgoPulse PTI 시리즈
   ```

2. **관련 상품 링크** (상세 페이지 하단)
   - 동일 카테고리 인기 상품 4개
   - 함께 구매하면 좋은 액세서리

3. **Footer 링크**
   - 전체 카테고리 구조
   - 고객지원 페이지
   - 회사 정보

4. **앵커 텍스트 최적화**
   - ❌ "여기 클릭", "더보기"
   - ✅ "에어 조립 도구 보기", "ErgoPulse 시리즈 상세정보"
